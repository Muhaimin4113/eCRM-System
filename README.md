# Customer Relationship Management (CRM) System

A web-based Customer Relationship Management (CRM) system developed as a Final Year Project to streamline business communications, manage internal workflows, and track sales performance.

---

## 🛠️ Core Technologies
* **Backend:** PHP (Modular structure)
* **Database:** MySQL (Optimized relational structure)
* **Frontend:** HTML5, CSS3, JavaScript
* **Design & Layout:** Responsive Web Design (Cross-device compatibility)
* **IDE:** Visual Studio Code

---

## 🚀 Key Modules & Features
* **Sales Tracking Dashboard:** Delivers data insights through a basic analytics dashboard for monitoring sales performance.
* **Internal Staff Forum:** A secure, staff-only communication hub built to manage and resolve customer complaints efficiently.
* **Responsive User Interface (UI):** Clean, accessible, and structured interface optimized for multi-device experience.
* **Data Security:** Features secure session handling and role-based access for system security.

---

## 📂 Project Structure Highlights
* Fully dynamic data handling using PHP and MySQL queries.
* Emphasized practical coding practices and proper database management system (DBMS) constraints.
